01 What is Dreams for Schools?

Dreams for Schools is a 501(c)(3) that oversees programs and initiatives that contribute to educating youth about Science, Technology, Engineering & Math fields. The main outreach program of the organization is the AppJam+ initiative.

Role: Digital Design Intern
Timeline: May 2020 - Current
Graphic Design, Website Design, Social Media Managment

02 Program Graphics

I wanted the graphics/icons to have a sense of cohesiveness when put together. I utilized the main colors from the brand guidelines as well as similar elements to achieve this.

05 CAKE Kit

CAKE stands for Cool Activities for Kids to Engineer.

The kit contains easy-to-follow instructions through an educational context that allows for an enjoyable learning experience. The kit provides each student with all of the materials necessary to complete each project. I was tasked with designing and laying out the booklet.
